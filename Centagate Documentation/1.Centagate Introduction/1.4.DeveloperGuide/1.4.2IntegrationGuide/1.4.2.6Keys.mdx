# 🔑Keys

In CENTAGATE context of keys, it served as unique identifiers used in encryption, decryption, database management, and access control. These keys are essential for security, data integrity, and efficient data management. Keys are used in encryption algorithms, database management systems for record identification, and access control mechanisms for authentication and authorization. There are 2 type of keys that are important during calling and API or integrations, which are:
- Integration Key
- Secret Key

### 🗝️Key Usage

Both keys are important when conduct and API call or any integration process. In order to get the keys, follow below steps to successfully retrieve the keys.


### 🤔💭Where to get Integration Key and Secret Key?
To get both keys, you may follow below steps. 

### Step 1️⃣:
To obtain both keys, the initial step involves registering an application within your existing account. Login to your admin account and from the navigation bar select **"Apps → Add App"**.

- ***User group allowed to access this app*** - Which group under the company have permission to use the app.
