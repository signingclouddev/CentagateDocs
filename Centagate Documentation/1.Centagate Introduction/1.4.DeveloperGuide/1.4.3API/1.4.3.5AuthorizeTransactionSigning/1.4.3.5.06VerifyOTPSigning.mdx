#  üïµ Verify OTP Signing

OTP Signing Verification is the function that verify OTP that generated by the transaction.

### üîç URL

| **METHOD**   | **PATH**                                                                      | 
| ------------ | ----------------------------------------------------------------------------- | 
| POST         | [host]/v2/CentagateWS/webresources/trans/authorise/[username]                 |


### üìèParameters
| KEY            | OCCURRENCES | DESCRIPTION                 |
| -------------- | ----------- | --------------------------- |
| username       | Required    | The user's username.        |
| otp            | Required    | The selected device account Id.|
| challenge      | Required    | The OTP challenge.          |
| details        | Optional    | The transaction details.    |
| ipAddress      | Optional    | User‚Äôs current ip address.  |
| userAgent      | Optional    | Platform information of authentication request is made of.|
| browserFp      | Optional    | User‚Äôs current browser fingerprint.|


::: tips
‚ö†Ô∏è Make sure to follow the correct spelling of the key.
:::

### üêç Sample Code in Python

```python
#Param values
USERNAME = {{username}}
CHALLENGE = {{challenge}}
OTP = {{otp}}
DETAILS = details}}
AUTHTOKEN = {{authToken}}
INTEGRATIONKEY = {{integrationKey}}
UNIXTIMESTAMP = int(time.time())
SUPPORTFIDO = {{supportFido}}
IPADDRESS = {{ipAddress}}
USERAGENT = {{userAgent}}
BROWSERFP = {{browserFp}}
SECRETKEY = {{secretKey}}


#calculate HMAC 
message = (USERNAME + CHALLENGE + OTP + DETAILS + INTEGRATIONKEY + str(UNIXTIMESTAMP) + AUTHTOKEN + SUPPORTFIDO + IPADDRESS + USERAGENT + BROWSERFP)

#convert to bytes
secret_key_bytes = bytes(SECRETKEY, 'utf-8')
message_bytes = bytes(message, 'utf-8')

#HMAC SHA-256
hash = hmac.new(secret_key_bytes, message_bytes, hashlib.sha256)
HMAC = hash.hexdigest()

conn = http.client.HTTPSConnection({{host}}, context=context)
payload = json.dumps({
  "username": USERNAME,
  "challenge": CHALLENGE,
  "otp": OTP,
  "details": DETAILS,
  "authToken": AUTHTOKEN,
  "integrationKey": INTEGRATIONKEY,
  "unixTimestamp": str(UNIXTIMESTAMP),
  "supportFido": SUPPORTFIDO,
  "ipAddress": IPADDRESS,
  "userAgent": USERAGENT,
  "browswerFp": BROWSERFP,
  "hmac": HMAC
})
headers = {
  'Content-Type': 'application/json'
}
conn.request("POST", "/v2/CentagateWS/webresources/trans/authorise/" + USERNAME, payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))
```
