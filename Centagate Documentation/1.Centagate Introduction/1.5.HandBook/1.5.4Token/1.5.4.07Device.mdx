# 📱Device

After successfully import the seed file to CENTAGATE, admin can now bind any token available to any user they need. Follow below steps to successfully bind a token to user.

### Step 1️⃣: 
From the navigation bar, select **Users → User List**.

### Step 2️⃣: 
From the user list, hover your cursor to **More → Manage OTP Token**.

&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/otptoken2.gif" 
    width="500" 
    height="200" 
    alt="OTP Token (2)"
  />
</p>


### Step 3️⃣: 
On the Manage OTP Token page, **click on the 🔍 icon**. A list of available tokens will be displayed. Select any token you prefer to bind. 

&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/otptoken3.gif" 
    width="500" 
    height="450" 
    alt="OTP Token (3)"
  />
</p>

### Step 4️⃣: 
Click **“Register” button** to bind the token to selected user. Wait until a success message prompt.

&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/otptoken4.gif" 
    width="500" 
    height="300" 
    alt="OTP Token (4)"
  />
</p>

Now token is successfully register under this user.
