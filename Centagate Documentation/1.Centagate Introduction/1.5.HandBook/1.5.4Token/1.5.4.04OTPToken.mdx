# 👤Add User

Administrators have the capability to add users to CENTAGATE. Follow below steps to successfully add users.

### Step 1️⃣: 
Login to CENTAGATE as **administrator**.

### Step 2️⃣: 
On the navigation bar, click **Users → Add User**.

### Step 3️⃣: 
Insert all field that has been marked with **asterisk (*) symbol**. 

Which are: 

- ***First Name*** - User’s first name.
- ***Last Name*** - User’s last name.
- ***Username*** - User’s username. This value **may not be changed** once it is set.
- ***Email*** - User’s email.
- ***Role*** - User’s role.
- ***Group Name*** - User’s group name.

### Step 4️⃣: 
Click **‘Save’** button to save.

>❗ If you are accessing CENTAGATE with a **GROUP ADMIN account**, please be aware that **you may not have** the authority to select the group name. This limitation exists because Group Admins consistently add new users under the same group. Your view may appear as shown below.
