# 🔃Update Token

For some certain reason, admin may need to update the token details of any users which maybe the users no longer use the token, or the user no longer exist to use the token. 
They might need to change or update the token details based on their need and preferences. 
To do so, company admin must know and acknowledge on the token type that exist and the usage. 
To successfully update the token details, admin can follow below steps.

> 🌟 To refer on token information, click [here](1.5.4.03BindToken.mdx) page to have a short read.

### Step 1️⃣: 
Login to CENTAGATE using any admin account.

### Step 2️⃣: 
From the user list, hover your cursor to **More → Manage OTP Token**.

### Step 3️⃣: 
On the Manage OTP Token page, **click on the 🔍 icon**. A list of available tokens will be displayed. Select any token you prefer to bind. 

&#x20;    
<p align="center">
  <img

  />
</p>

### Step 4️⃣: 
Click **“Register” button** to bind the token to selected user. Wait until a success message prompt.
