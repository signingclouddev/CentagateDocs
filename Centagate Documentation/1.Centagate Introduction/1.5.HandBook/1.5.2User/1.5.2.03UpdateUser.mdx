# 🔃Update User

After successfully adding users to CENTAGATE, administrators may find it necessary to update user information periodically to ensure accuracy. To perform updates on user details, follow the steps below.

### Step 1️⃣: 
On the navigation bar, click **Users → User List**.
&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/update_1.gif" 
    width="300" 
    height="400" 
    alt="Update User (1)"
  />
</p>

### Step 2️⃣: 
In the user list, locate the 'Action' button, and within its options, you will find the **'Update'** button. Click on it to proceed.

&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/update_2.jpg" 
    width="600" 
    height="300" 
    alt="Update User (2)"
  />
</p>

### Step 3️⃣: 
On the 'Update User' page, feel free to modify any details, but ensure that no fields marked with an asterisk (*) symbol are left empty.

&#x20;    
<p align="center">
  <img 
    src="https://raw.githubusercontent.com/signingclouddev/CentagateDocs/main/image/1.5 Hanbook/update_3.png" 
    width="350" 
    height="500" 
    alt="Update User (3)"
  />
</p>

>❗ If you are accessing CENTAGATE with a **GROUP ADMIN account**, please note that **you may not** have the authority to change or update the user's group. Your view may appear as follows.
