**üí°Password Settings**
Under this section, you have to provide:

**üí°Session Settings**
In this section, you need to provide:   
- ***Allowed login attempts*** - A specified number of login attempts is allowed for each user. If the attempts exceed this limit, the user's account will be locked.
- ***Lock effective period*** - Duration for the user's account to recover from a locked status.
- ***Session timeout*** - Session timeout duration for each user.
- ***Dormant period*** - The user account will enter a dormant status if there is no login activity for a specified period.
- ***Maximum number users per session*** - Maximum number of users allowed to log in per session.
