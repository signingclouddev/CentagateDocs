# 📱 Mobile or 🌐 Web API

To register mobile device protocol into CENTAGATE, follow below steps:

### Step 1️⃣: 
Login to CENTAGATE using admin account.

### Step 2️⃣: 
From the navigation bar, select ***“Apps → Add Apps”***.

### Step 3️⃣: 
In the Add App form, 2️⃣ section you need to provide. 

<br>

### 💡 General Info
In the general section, you may need to provide:
- ***App name*** - The apps name.
- ***Domain Name*** - Domain to access the app.
- ***Status*** - The status of the app either ACTIVE or INACTIVE.
- ***User groups allowed to access this app*** - The selected group that can access to the app.

### 💡 Integration Details
Under this section, only 2️⃣ fields to be provide.  Which are,
- Integration type -> Choose “Mobile” or Web API for this field.
- Allowed IP addresses -> Provide allowed IP address to access to this type of protocol.

Click ***“Save"*** button to save the details.
