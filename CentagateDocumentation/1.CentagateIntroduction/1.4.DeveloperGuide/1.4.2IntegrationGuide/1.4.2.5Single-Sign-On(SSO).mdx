# 🔐 Single-Sign-On (SSO)

Single sign-on in a **CENTAGATE Cloud** is an authentication scheme that allows a user to log in with a single ID and password together with two factor authentications which provided by **CENTAGATE Cloud**.  <br>

True single sign on allows the user to login once and access services without re-entering authentication factors. **CENTAGATE Cloud** acts as Identity Provider and provide connection with other service provider such as *Office365,  dropBox, Gsuite, Gitlab, Github*, and etc.

To start integrating using SSO, you may follow below steps.

### Step 1️⃣: 
Login to CENTAGATE using a company admin account.

### Step 2️⃣:
From the navigation bar, click ***“SSO Management → Add SSO Integration”***.

### Step 4️⃣:
After clicking the icon, you may need to provide several fields. Which are: 
- ***Name*** - Name of the App. <br>
- ***Unique ID*** - The unique ID. <br>
- ***Group allowed*** - Group allowed to use SSO using the app.
